What is join?

Join is a Website for a Kanban. Kanban is a visual project management system that utilizes a board with columns to represent work items. Its main objectives are optimizing workflow, limiting work in progress, and promoting continuous improvement. Kanban enables teams to track progress, identify bottlenecks, and adapt to changes effectively. By fostering efficiency, waste reduction, and collaboration, it has become popular in agile and lean methodologies, driving successful project outcomes.

How to use it

  1. In our summary page, you will find an overview of the status of various tasks. Additionally, the next deadline is displayed. Clicking on a card will redirect you to the board page.
  2. The "Add Task" feature allows you to create new tasks. You can set the title, add descriptions, choose a category, assign the task to a person, define the due date and priority.
    If needed, you can also add subtasks. If the desired category doesn't exist, you can create a new category.
    You can access task creation through our board page or contacts page.
  3. On our board page, you can assign the appropriate status (to do, in progress, awaiting feedback, done) to existing tasks. You can do this by dragging and dropping tasks or by clicking on the arrow next to the color-coded category.
    It is also possible to edit existing tasks on this page.
    Additionally, you can use the search bar to filter existing tasks.
  4. Under "Contacts", you have access to your contacts. Here, you can create new contacts or edit existing ones.
    By clicking on a person in the contact list, the contact's information will be displayed, including all saved data. This is also where you can make further edits to the contact.
